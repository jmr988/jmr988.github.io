Project Overview

I led the end-to-end MVP for the APS Training Academy mobile app, enabling parents to browse courses, pass a short eligibility check, create an account, register children, and book classes. I also delivered admin tools for schedules and scholarship workflows, plus SEO-optimized program sites to support recruitment.

Role: Project Manager & Software Developer • Timeline: 4 weeks • Company: APS Data Technologies

Key Features

  • Course browsing with schedule, location, and capacity.
  • 3-question eligibility check → guided account & child registration.
  • Bookings with reminders; past & future classes.
  • Admin tools for schedules, locations, scholarships, and rosters.
  • Two SEO-optimized program sites for recruitment.

Tech Stack

React Native
Firebase Auth & Realtime Database
WordPress + Elementor Pro (SEO sites)
Git & Asana (collaboration)

Outcome & Impact

  • MVP validated with stakeholders; clear path for v1 features and data model upgrades.
  • Admin workflows reduced manual steps for scholarship approvals and scheduling.
  • Public sites improved clarity and lead capture for recruitment.
